# Jogos em Rede Multiusuário

Este repositório começa pela [Aula Zero](./aulas/aula-zero/aula-zero-websocket.md), uma introdução a `WebSocket` com um chat em modo texto.

Para executar:

```bash
cd aulas/aula-zero
npm install
npm run server
```

Em outro terminal (um pra cada usuário no chat):

```bash
cd aulas/aula-zero
npm run client
```
